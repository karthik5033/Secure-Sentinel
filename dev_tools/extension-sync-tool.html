🛡️ SecureSentinel Extension Sync Tool

Use this tool to force sync the blocklist and test blocking.

📋 Instructions

  1. Reload Extension: Go to chrome://extensions and click reload
  2. Check Status: Click "Check Extension Status" above
  3. Force Sync: Click "Force Sync Blocklist"
  4. Test: Try visiting a blocked domain

🔧 Manual Commands

Open extension console (chrome://extensions → service worker) and run:

// Check blocklist
console.log(Array.from(permanentBlocklist));

// Force sync
syncBlocklist().then(() => console.log("Synced:", permanentBlocklist.size));
